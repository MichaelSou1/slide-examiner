# Part 1 — S-group (semantic) detection on 30B-A3B

Real VLM probe (vLLM, Qwen3-VL-30B-A3B-AWQ) over the S-group subset (S1-S6 + 40 clean negatives, 104 samples), modalities A/B/B'/C, task T1. B' captions are generated by the same model describing each rendered image (`scripts/caption_images.py`). 0% parse failure.

## Findings — semantics is the examiner's job (mirror image of geometry)

- **30B detects the semantic group at 50-100%**, versus ~0 for fine geometry (`reports/part1_geometry_threshold.md`). The examiner's value is the S-group, exactly as the contract intends.
- **S1 title/body mismatch reads straight off the image** (A 8/8 = 100%), and **S5 missing-section** is caught ~88%. These are genuine multimodal reasoning wins, not structure look-ups.
- **For deck-level semantics the natural-language caption oracle (B') is the *strongest* channel** (deck-S B' = 20/24 = 83%, vs C = 11/24). This is the exact opposite of geometry, where B' was a dead channel — it validates the B'-via-VLM-captioning fix: S2 narrative-order is 8/8 from B' (the joined per-page captions expose the shuffled order) and S3 terminology is best from B' too.
- **Stacking page images (C) consistently underperforms** the best single channel for deck semantics (C 11/24 < B' 20/24) — the multi-image deck request, though now wired, distracts the model; the structured/linguistic summary is what carries cross-page reasoning.
- **S6 image-text contradiction is 0 everywhere** because the synthetic corpus has no image elements — the defect is not manifest. A real S6 arm needs image-bearing decks; flagged for the dataset freeze.
- **Mild false positives appear on the image channel** (A 3/40) where geometry had none; B/B'/C stay at 0 FP. 30B is slightly more willing to over-call semantics from pixels.

## Implication

Part 1's division of labor is now empirically grounded on a 30B model: **G1-G6 -> symbolic linter** (VLM blind), **S1-S6 -> examiner**, with the language/structure channels (B, B') carrying deck-level reasoning better than raw multi-image. The full matrix should weight S-group coverage and the B/B' arms, keep C for page-level, and add image-bearing decks so S6 becomes testable.
